- 실시간 영상 송출 ( Kurento 사용 )

- 실시간 채팅 / 1:1 채팅(귓속말)

- **시청자 수**
- **방송을 켠 순간부터 모금된 금액**
- **방송시간**
- **영상 저장 없고, 채팅 기록 저장 없고, 후원만 저장**
