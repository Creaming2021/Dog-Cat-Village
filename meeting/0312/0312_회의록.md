# 회원가입

- 전화번호 인증 번호 없애고 이메일 입력받아 인증

    ⇒ 가입완료 완료 후에 이메일 확인하라고 alert 보여주고

    ⇒ 처음 가입하는 유저의 경우 메일에서 완료 링크 누르면 그 다음부터 로그인 가능 

    ⇒ 이미 존재한 유저면 가입했다고 이미 존재하는 유저라고 alert로 보여줌 ( 인증 유무 둘 다 )

- 전화번호는 그냥 정보 저장용으로 입력 받음
